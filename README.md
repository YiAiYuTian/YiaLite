# YiaLite
YiaLite is a lightweight and easy-to-use development library, built on top of mature open-source components such as SDL3, covering core functionalities including window management, input events, 2D graphics rendering, and audio playback.

## Third-Party Libraries
* [SDL3](https://github.com/libsdl-org/SDL)
* [spdlog](https://github.com/gabime/spdlog)
* [Vorbis](https://github.com/xiph/vorbis)
* [MiniAudio](https://github.com/mackron/miniaudio)
